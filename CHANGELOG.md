# 更新日志

## [v1.0.1] - 2025-12-01

### 新增
- 改进 CI/CD 流程，支持多架构 macOS 版本 (Intel 和 ARM)
- 添加测试作业到发布工作流
- 使用更安全的 GitHub Actions 权限模型

### 更改
- 重构 release.yml 工作流，优化作业依赖顺序
- 更新 GitHub Actions 到最新版本
- 改进构建脚本以支持多架构构建
- 从 CHANGELOG.md 自动生成发布说明

### 修复
- 修复作业依赖顺序问题，确保先构建再发布

## [v1.0.0] - 2025-11-26

### 新增
- 初始化项目，实现基本的 AI 模型访问代理功能
- 支持多种 AI 服务提供商（Ollama, DeepSeek, Aliyun, OpenRouter）
- 实现 OpenAI 兼容 API 接口
- 支持模型配置和自定义请求头处理
- 实现完整的请求和响应日志记录
- 支持 HTTPS 证书验证配置
- 实现跨平台构建（macOS/Linux/Windows）

### 更改
- 使用 TypeScript 重构代码，提高类型安全性
- 模块化设计，便于维护和扩展
- 使用 pnpm 作为包管理工具
- 使用 Bun 构建跨平台独立二进制文件

### 修复
- 修复了请求头处理中的潜在问题
- 优化了错误处理机制
- 改进了日志记录功能
